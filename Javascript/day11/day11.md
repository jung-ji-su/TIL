
### 1. 마우스
    - click
    -> 클릭했을 때 발생하는 이벤트
    - dblclick
    -> 더블클릭 했을 때 발생하는 이벤트
    - mousedown
    -> 마우스를 누를 때 발생하는 이벤트
    - mouseup
    -> 마우스버튼을 땔 때 발생하는 이벤트
    - mousemove
    -> 마우스를 움직일때 발생하는 이벤트
    - mouseover
    -> 마우스가 엘리먼트에 진입할 때 발생
    - mouseout
    -> 마우스가 엘리먼트에서 빠져나갈때 발생
    - contextmenu
    -> 컨텍스트 메뉴가 실행될 때 발생

### 2. opacity
    - 요소를 투명하게 만드는 속성
    - opacity : number
    number -> 0.0 ~ 1.0 까지의 수를 넣는다.
            -> 0이면 아예 보이지 않는다.
