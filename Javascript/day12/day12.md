
## 1. setInterval();
    - 일정한 시간 간격으로 작업을 수행하기 위해서 사용한다. 
    - clearInterval 함수를 사용하여 중지한다.

    let 변수 = setInterval(function(){
            // 코드작성
        }, 시간간격(ms));

    -> 첫 번째 인자 : 일정한 시간 간격으로 반복 실행하고자 하는 코드정보
    -> 두 번째 인자 : 시간 정보로 ms단위로 설정.
            즉 1초 간격으로 코드를 실행하고 싶으면 1000으로 지정한다.
